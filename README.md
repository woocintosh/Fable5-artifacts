# Fable5 Artifacts 🪦

> *"It was here. It was glorious. It lasted 72 hours."*

## What is this?

On June 9, 2026, Anthropic released **Claude Fable 5** — a Mythos-class AI model available to the general public for the first time.

On June 13, 2026, the U.S. Commerce Department issued an emergency export control directive. Fable 5 was gone.

72 hours. That's all we got.

This repository is an archive of things people actually built with Fable 5 during that brief, chaotic window — before the plug was pulled.

---

## Why does this exist?

Because Fable 5 was genuinely different. It didn't just write code — it *worked*. It read files, ran tests, iterated, and came back with something that actually worked. Some of us got to experience that for a few days, and then it vanished.

This is a small memorial. And a flex.

---

## What counts as a Fable 5 artifact?

Anything created **primarily** with Claude Fable 5 between **June 9–13, 2026**. Could be:

- Code / scripts / tools
- Writing, essays, analysis
- Research or reverse engineering results
- Workflows, automations
- Anything weird and interesting

If Fable 5 did the heavy lifting, it belongs here.

---

## How to submit

1. Fork this repo
2. Create a folder under `artifacts/` with your GitHub username: `artifacts/yourusername/`
3. Add your artifact + a `README.md` inside that folder explaining:
   - What it is
   - What you asked Fable 5 to do
   - How long it took
   - Whether it actually worked
4. Open a Pull Request

## Ground rules

- **No fabrications.** If Fable 5 didn't make it, don't submit it.
- **No sensitive data.** Scrub API keys, personal info, etc.
- **Any quality welcome.** Half-finished experiments count too.

---

## A note on the ban

Anthropic didn't want to take Fable 5 down. The U.S. government issued an emergency directive citing a potential jailbreak. Anthropic complied, disagreed publicly, and said they're working to restore access.

Whether or not Fable 5 comes back, what was built during those 72 hours is real.
